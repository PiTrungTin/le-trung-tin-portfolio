# Le Trung Tin Portfolio

Personal portfolio site for Le Trung Tin, built with React, TypeScript, Vite, TailwindCSS, Framer Motion, and React Three Fiber.

The project is evolving from a 3D cyberpunk concept into a cleaner portfolio focused on presenting backend and full-stack engineering experience clearly for recruiters and hiring teams.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Current Direction

- English-language recruiter-facing portfolio
- Resume-driven content based on real experience
- Bright, premium visual direction with lighter motion and reduced visual noise
- Ongoing simplification away from scaffold defaults and placeholder branding
