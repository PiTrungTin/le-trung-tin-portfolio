# Le Trung Tin Portfolio — Implementation Plan

## Context

This repository started as an experimental 3D cyberpunk portfolio and is being reshaped into a clearer recruiter-facing portfolio for Le Trung Tin.

The current product direction is:

- English-language portfolio
- Focus on backend and full-stack roles
- Bright, premium visual system
- Simpler, less distracting interaction model

## Current Stack

- React
- TypeScript
- Vite
- TailwindCSS v4
- Framer Motion
- React Three Fiber
- Zustand

## Near-Term Goals

1. Replace scaffold metadata and placeholder branding with project-specific naming.
2. Improve readability, spacing, and section flow across the page.
3. Reduce visual noise and move away from effect-heavy presentation.
4. Align content with real resume data and recruiter expectations.

## Planned Content Structure

- Hero
- About
- Skills
- Experience
- Contact
